プロジェクト

① 日本語チャットボット (Flask + NLTK)

Flask と NLTK を使用した日本語対応のシンプルなチャットボットです。ユーザーの入力を分析し、基本的な挨拶や質問に対して応答します。 UI はダークモード対応で、ボットのアバター表示やタイピングアニメーションも実装済み。会話履歴は JSON に保存され、将来的な分析に活用可能です。

  • NLTKで日本語トークン化
  • Flaskフォーム連携
  • 会話ログ保存（JSON）
  • ダークテーマ＋アニメーション効果

② 感情認識Webアプリ (Flask + TensorFlow)

ユーザーが画像をアップロードすると、感情分類モデル（FER2013ベース）を使って顔の表情を7カテゴリに分類（怒り・嫌悪・恐れ・幸せ・悲しみ・驚き・無表情）。 結果は UI 上に表示され、同時にCSVファイルとして履歴に記録されます。Tailwindを使ったダークテーマやスピナーによる「感情を分析中...」表示も実装済み。

  • 画像分類モデルをTensorFlow/Kerasで自作
  • FER2013のデータから訓練
  • リアルタイムアップロード＆応答
  • 履歴CSV保存機能付き
